üîç Pattern-Based Labeling

The Advanced Labelization panel allows you to find and label multiple occurrences efficiently:

  1. Enter a text pattern in the search field
  2. Navigate through matches with Previous/Next buttons
  3. Select portions of the match and apply labels
  4. Use "Apply" for the current match or "Apply All" for all occurrences
üí° Pro Tip
Build complex nested labels once, then apply to all similar patterns in your document with a single click.

üîó Group Management & Coreference Resolution

Link related annotations together to represent the same entity mentioned multiple times:

How it Works

  • Group ID: Use the same identifier (e.g., docid) for related mentions
  • Group Attributes: Designate which attributes should sync across all group members
  • Active Groups Panel: View all coreference groups in real-time
  • Automatic Sync: Changes to group attributes propagate to all linked labels

Example Use Case

If a document references "IRPA", "the Act", and "Immigration and Refugee Protection Act", you can give all three the same docid="irpa" to indicate they refer to the same legislation.

üìñ Usage Examples

Example 1: Basic Legal Citation

Original text:

See Theratechnologies inc. v. 121851 Canada inc., 2015 SCC 18

After labeling:

<manual_label labelname="mention" docid="scc2015-18" doctype="decision">
  <manual_label labelname="title" titletype="case name">
    Theratechnologies inc. v. 121851 Canada inc.
  </manual_label>, 
  <manual_label labelname="reference">
    2015 SCC 18
  </manual_label>
</manual_label>

Example 2: Coreference

Label multiple references to the same document with a shared docid:

<manual_label labelname="mention" docid="irpa">Immigration and Refugee Protection Act</manual_label>
...
<manual_label labelname="mention" docid="irpa">IRPA</manual_label>
...
<manual_label labelname="mention" docid="irpa">the Act</manual_label>

üîß Technical Details

Data Storage

Labels are stored as HTML comments before the <head> tag:

<!-- HTMLLabelizer {"labels": [...], "version": "1.0"} -->

Applied Labels

Annotations use custom <manual_label> elements with attributes:

<manual_label labelname="mention" docid="example" doctype="legislation">
  text content
</manual_label>

Browser Compatibility

  • Chrome/Edge 90+
  • Firefox 88+
  • Safari 14+

No Installation Required

The tool is a static web application that runs entirely in your browser. No server, no installation, no data transmission.
